# Aloura



Aloura is a MVC Appication Framework, for building real-time, web, mobile, desktop, console, and iot appications. 	



# Stack



* Angualrjs

* Koa

* RethinkDB

* Socket.io

* Electron (Windows, Mac, Linux)

* Cordova (iOS)

* Crosswalks (Android)

* Universal Windows Platform (Windows Store)  



# Features



* Admin Control Panel

* CLI

* User Auth

* Realtime
